 Traffic Data Analysis Dashboard

This project analyzes keyword traffic performance using Power BI and MySQL
It includes an optimized star-schema data model and an interactive dashboard to monitor:

- Total Traffic & Traffic Cost
- Search Volume Trends
- Keyword Difficulty
- Traffic % by Keyword Group
- CPC Insights

 Key Features:
- MySQL relational data model (lookup & fact tables)
- Power BI dashboard with interactive visuals
- Clean ETL process for keyword analytics
- Charts: Bar, Line, Pie, KPI cards, Tables

Tools Used:
- Excel
- Power BI
- MySQL Workbench
- SQL
